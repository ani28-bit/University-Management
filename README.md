# University Management System

A Java-based University Management System developed to manage university operations efficiently. The system provides separate functionalities for administrators, teachers, and students, including course management, student registration, marks entry, GPA calculation, and result publishing.

## Features

* Student registration and management
* Teacher registration and management
* Course and subject management
* Teacher-course assignment
* Marks entry by teachers
* GPA calculation
* Result generation and publishing
* User-friendly Java Swing graphical interface
* Database integration using MySQL

## Technologies Used

* Java
* Java Swing (GUI)
* JDBC
* MySQL
* IntelliJ IDEA

## Project Structure

```text
src/
├── admin/
├── student/
├── teacher/
├── database/
├── models/
└── utils/
```

## Setup Instructions

1. Clone the repository:
   git clone https://github.com/ani28-bit/UniversityManagementSystem.git

2. Open the project in IntelliJ IDEA.

3. Create a MySQL database and import the provided SQL file.

4. Configure the database connection parameters in the project.

5. Build and run the application.
